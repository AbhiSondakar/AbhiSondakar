## 🚀 Featured Project

### [NLP Conversational Assistant to Database](https://github.com/AbhiSondakar/text-to-sql)

> A system that allows users to query databases using natural language instead of SQL.

**Technologies** &nbsp;
<img src="https://skillicons.dev/icons?i=python" height="20"/> Python &nbsp;
<img src="https://skillicons.dev/icons?i=postgres" height="20"/> PostgreSQL

**Key Features**
- 🔄 Converts natural language queries into SQL
- 🤖 Dual-agent architecture (Generator + Validator)
- 📊 Automatic visualization of query results
- 🛡️ SQL validation layer to prevent destructive queries
- ⚡ LLM fallback with retry and circuit breaker logic
